# junco-site Dark Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the approved dark redesign — dark-only theme, full-viewport glyph-matrix canvas background, and a naturalist dark-eyed-junco mascot across all site touchpoints — with every content gate green.

**Architecture:** Pure static site (no build step at deploy). One new dependency-free script `site/glyphs.js` paints a fixed background canvas. All colors flow from CSS custom properties defined only in `site/styles.css` token blocks. Docs pages are generated by `scripts/build-docs.mjs` and committed; any template change requires regenerating and committing `site/docs/**`.

**Tech Stack:** Vanilla HTML/CSS/JS · Node ≥ 22 (`scripts/build-docs.mjs`) · Playwright via `npx` for screenshots/PNG regeneration · `gh` for the PR.

**Spec:** `docs/superpowers/specs/2026-07-22-dark-glyph-redesign-design.md`

## Global Constraints

- Work happens on branch `feat/dark-glyph-redesign`; `main` is PR-only (quality-gate ruleset).
- **Banned-words gate:** `grep -rniE 'blazing|seamless|revolutionary|supercharge|magical|\beasy\b|\bsimply\b|powerful' site/ og.html` must exit 1.
- **Vendor gate:** `grep -rniE 'anthropic|claude|gpt|gemini|llama|mistral|deepseek|qwen|ollama|vllm|lm.?studio|mlx' site/ og.html .github/ --exclude=quality-gate.yml` must exit 1.
- **Hex gate (HTML):** `grep -rn '#[0-9a-fA-F]\{3,8\}\b' site/ --include='*.html'` may hit ONLY `theme-color` metas (after this work: exactly one per page). All SVG fills/stops in HTML use `var(--token)` or `currentColor`.
- **Hex gate (CSS):** 6-digit hex in `site/styles.css` only inside `:root` and `@media print`; `site/docs/docs.css` stays hex-free.
- **Emoji gate:** only ✓ and ✗ allowed in gated files; the glyph pool for the canvas lives in `site/glyphs.js` as escaped ASCII/`·`/`•` (not a gated file, but keep it clean anyway).
- **Word gate:** ≤ 450 visible words on `site/index.html` (script/style/pre/dl/visually-hidden stripped). Expected after this work: **449**.
- **Docs gate:** `node scripts/build-docs.mjs --check` green; after any template edit, run `node scripts/build-docs.mjs` and commit regenerated `site/docs/**`.
- No new visible words from any mascot/canvas markup: decorative SVGs are `aria-hidden="true"` with no text nodes.
- Local preview for anything visual: `python3 -m http.server 8123 -d site` (docs pages need `/`-rooted asset paths, so preview over HTTP, not `file://`).

```js
// glyph-matrix background — magicui's glyph-matrix (React/canvas) redone as a
// dependency-free script. A fixed canvas behind the page; a few cells mutate
// per tick. Static single frame under prefers-reduced-motion; absent without JS.
(function () {
  var GLYPHS = "01·•+*/\\<>=";
  var CELL = 14; // px
  var RATE = 0.04; // share of cells mutated per tick
  var TICK = 90; // ms

  var canvas = document.createElement("canvas");
  canvas.id = "glyphs";
  document.body.prepend(canvas);
  var ctx = canvas.getContext("2d");

  var reduced = matchMedia("(prefers-reduced-motion: reduce)");
  var cols, rows, timer;

  function cell(i) {
    var x = (i % cols) * CELL;
    var y = ((i / cols) | 0) * CELL;
    ctx.clearRect(x, y, CELL, CELL);
    ctx.globalAlpha = 0.3 + Math.random() * 0.7;
    ctx.fillText(GLYPHS[(Math.random() * GLYPHS.length) | 0], x + CELL / 2, y + CELL / 2);
    ctx.globalAlpha = 1;
  }

  function build() {
    var dpr = devicePixelRatio || 1;
    canvas.width = Math.ceil(innerWidth * dpr);
    canvas.height = Math.ceil(innerHeight * dpr);
    ctx.setTransform(dpr, 0, 0, dpr, 0, 0);
    ctx.fillStyle = getComputedStyle(document.documentElement).getPropertyValue("--glyph").trim();
    ctx.font = '11px "Commit Mono", monospace';
    ctx.textAlign = "center";
    ctx.textBaseline = "middle";
    cols = Math.ceil(innerWidth / CELL);
    rows = Math.ceil(innerHeight / CELL);
    for (var i = 0; i < cols * rows; i++) cell(i);
  }

  function tick() {
    var n = Math.max(1, (cols * rows * RATE) | 0);
    for (var k = 0; k < n; k++) cell((Math.random() * cols * rows) | 0);
  }

  function start() {
    if (!timer && !reduced.matches && !document.hidden) timer = setInterval(tick, TICK);
  }
  function stop() {
    clearInterval(timer);
    timer = null;
  }

  build();
  start();
  if (document.fonts) document.fonts.ready.then(build); // repaint once the woff2 lands
  addEventListener("resize", build);
  document.addEventListener("visibilitychange", function () {
    if (document.hidden) stop();
    else start();
  });
  reduced.addEventListener("change", function () {
    stop();
    build();
    start();
  });
})();
```
